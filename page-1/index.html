What is Three.js?

  • Three.js is a JavaScript library that makes it easy to create 3D graphics in a web browser.
  • It has a fairly steep learning curve
  • But the alternative is to use pure WebGL—a waaaaaay steeper learning curve!
  • Once you get the hang of it, it makes creating interactive 3D scenes possible—nay, enjoyable!
  • Let’s look at some examples of what you can make using Three.js!!!
